A simple **Paypal Express Checkout** plugin for a direct payment. Interaction with **WP Availability Calendar & Booking** plugin 

== Description ==
The short code [paypal] adds a payment form to the page.

There are two sections of settings, one for the Paypal Sandbox and one for the Real Life use.

When the Sandbox is selected and the other fields are emty the default Sandbox environment of PayPal is used. 
If you want to test with the Sandbox environment of your own account follow the Paypal instructions to obtain the Username, Password and Signature.

The Description option allows you to have a free format description or a Checkin Date with the format yyyy-mm-dd. The Checkin Date is required for interaction with the **WP Availability Calendar & Booking** plugin

== Installation ==

1. Upload the entire jm_paypal_client folder to the /wp-content/plugins/directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the settings in Admin->settings->WP Availability & Bookings Settings
